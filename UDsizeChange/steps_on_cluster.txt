# > 1. on how to run it on the cluster?
# After getting the image, you can get a shell inside the container with
apptainer shell geospatial_latest.sif
# Or you can launch R interactively with
apptainer exec geospatial_latest.sif R
# Or you can run a script with
apptainer exec geospatial_latest.sig Rscript foo.R

# > 2. how to get my data into it and out?
# Your homefolder should already be mounted at /u/<your_username> and there you should also have rw access. otherwise you can also mount other locations with the the --bind flag, e.g., in case you also need ptmp that would be
apptainer exec --bind /ptmp/<your_username> geospatial_latest.sif ...

# > 3. and how to submit jobs on it?
# You can use the same submission script as you did without apptainer. You just need to additionally load the apptainer module and instead of launching R directly you would call apptainer, i.e.,

module load apptainer/1.2.2
srun apptainer exec --bind <some_additional_mount> geospatial_latest.sif <your commands here>
